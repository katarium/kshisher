## 🚀 Instalación

### Requisitos Previos
- Linux/Termux/Android (Root recomendado)
- Conexión a internet
- Git instalado

### Instalación Rápida

```bash
# Clonar el repositorio
git clone https://github.com/katarium/kshisher

# Entrar al directorio
cd kshisher

# Dar permisos de ejecución
chmod +x kshisher.sh

# Ejecutar
bash kshisher.sh
